# terraform-aws-aks
### This module will help you to create AKS (Azure Kubernetes Services)

### Create provider.tf and paste the following content:
```
provider "azurerm" {
  version = "=2.0.0"
  features {}
}
```

### Create a file named: main.tf with the content below:
```
module "aks" {
    cluster_name                            =   "example-aks1"
    kubernetes_version                      =   "1.18.4"
    node_pool_name                          =   "node1"
    min_count                               =   "3"
    max_count                               =   "5"
    client_id                               =   ""
    client_secret                           =   ""
    environment                             =   "dev"
    resource_group_name_location            =   "West Europe"
    resource_group_name                     =   "dev"
    username                                =   "centos"
    vm_size                                 =   "Standard_A2_v2"
}
```
